## 🚀 Installation rapide

### Sur votre téléphone Android

1. **Télécharger l'APK** sur votre téléphone
2. **Activer l'installation depuis sources inconnues**:
   - Ouvrir Paramètres
   - Aller dans Sécurité (ou Applications)
   - Activer "Sources inconnues" ou "Installer des apps inconnues"
3. **Ouvrir le fichier APK** téléchargé
4. **Appuyer sur "Installer"**
5. **Lancer NeyshaPlay!** 🎊

### Via ADB (pour développeurs)

```bash
adb install NeyshaPlay-v1.0.0-release.apk
```

## 🔧 Spécifications techniques

### Compatibilité
- **Android minimum**: 5.1 (API 22)
- **Android recommandé**: 13+ (API 33)
- **Taille**: 3.6 MB
- **Permissions**: Caméra, Microphone, Stockage, Internet

### Technologies
- Next.js 15 (export statique)
- Capacitor 8 (wrapper natif)
- Firebase (backend)
- Cloudinary (vidéos)

## 📝 Notes importantes

### Première utilisation
1. L'app affiche un gate d'âge (18+)
2. Connexion Firebase requise
3. Sélection du rôle (Créateur/Utilisateur)
4. Permissions caméra/micro à accepter

### Connexion Internet
L'application nécessite une connexion Internet pour fonctionner.

### Données
- Les vidéos sont hébergées sur Cloudinary
- Les données utilisateur sur Firebase Firestore
- Authentification via Firebase Auth

## 🐛 Problèmes connus

1. **Commentaires**: Interface présente mais sauvegarde non implémentée
2. **Profils publics**: Route `/u/[id]` désactivée temporairement
3. **Settings**: Page désactivée pour ce build

Ces fonctionnalités seront ajoutées dans la v1.1.0.
