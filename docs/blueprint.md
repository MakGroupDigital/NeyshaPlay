# **App Name**: NeyshaPlay

## Core Features:

- Video Feed: Endless vertical scrolling feed of short videos.
- Video Player: Autoplay videos with sound control and swipe to creator profile.
- Content Creation: Direct access to the phone's camera with simple editing tools.
- Discover: Explore trending hashtags, popular videos, and categories.
- Notifications: Activity feed for new followers, likes, comments, and mentions.
- User Profiles: Display user profile, statistics, and published videos.

## Style Guidelines:

- Primary color (background): Deep black (#121212) to create a premium, immersive atmosphere.
- Accent color: Neon green (#8AFF00) for interactive elements and actions.
- Primary text color: Off-white (#E0E0E0) for titles and user names.
- Secondary text color: Medium gray (#A0A0A0) for descriptions and metadata.
- Font: 'Inter' sans-serif font for all UI elements.
- Titles (H1, H2): 'Inter' Bold.
- Body text: 'Inter' Regular.
- Buttons & UI: 'Inter' Medium.
- Minimalist outline icons; active icons turn neon green.
- Dark Mode only.
- Cards and modules with slightly rounded corners (8px to 12px border-radius).
- Subtle 'glow' effect with the neon green color (#8AFF00) on important elements.
